Мои тендеры

Изменить отрасли и регионы
 
 
  • Простой
  • Расширенный
  • =слово
    искать слово по точному совпадению
    =СИЗ  Исключит из результатов поиска «СИЗО»
  • ={словосочетание}
    искать словосочетание по точному совпадению
    ={КИП ИА}  Найдет точное словосочетание с пробелом, исключит из поиска «кипятильник»
  • !слово
    не обрезать окончание
    Вместо !слово можно использовать слово*. Результат будет одинаковый
    перевозка !сырья  Исключит из поиска «перевозка сыра»
  • слово~слово
    искать слова рядом в любой последовательности
    ремонт~полов  Исключит из поиска «ремонт машины для мытья полов»
  • слово<слово
    соблюдать последовательность слов
    Тех<осмотр  Найдет только те тендеры, в которых слово «тех» расположено до слова «осмотр» без учета расстояния между ними. Исключит поиска «осмотр сотрудников техникум
  • сложные формулы
    операторы можно комбинировать
    ремонт<спец~техники  Найдет: "услуги по ремонту спец техники", "ремонтные работы: автомобильная техника специальная", исключит из поиска: "поставка запасных частей для специальной техники (ремонтные комплекты)"
  • слово~N~слово
    искать слова на максимально допустимом расстоянии, где N количество слов
    ремонт~2~дорог  Найдет "ремонт автомобильных дорог", «ремонт покрытия автомобильной дороги», исключит из поиска «ремонт тротуаров вдоль автомобильных дорог общего пользования»
